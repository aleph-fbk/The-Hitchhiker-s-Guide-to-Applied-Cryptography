\subsection{Cryptosystems}\label{sec:cryptography_basics:cryptosystems}

A \textbf{cryptosystem} is an ordered list of ciphers along with the corresponding set of finite possible plaintexts, finite possible ciphertexts, and finite possible keys. There are two main types of cryptosystems: symmetric (\Cref{sec:symmetric_cryptography}) and asymmetric (\Cref{sec:asymmetric_cryptography}).
